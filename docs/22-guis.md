# GUI System

## Overview

The GUI system provides interactive inventory-based interfaces for managing kingdoms and viewing information.

## Key Components

### GUI Classes
- `KingdomManagementGUI.java` - Main kingdom management
- `ChallengeGUI.java` - Challenge viewing
- `ClaimMapGUI.java` - Interactive claim map
- `StatisticsDashboardGUI.java` - Statistics dashboard
- `TerritoryHeatMapGUI.java` - Territory heat map
- `MailGUI.java` - Mail inbox
- `EventCalendarGUI.java` - Event calendar
- `VaultGUI.java` - Kingdom vault

## Core Features

### Kingdom Management GUI
- **View Members**: List all members with roles
- **View Claims**: List all claimed chunks
- **View Flags**: Show kingdom and plot flags
- **View Statistics**: Display XP, level, contributions
- **Quick Actions**: Click items for actions

### Challenge GUI
- **Challenge List**: View all challenges
- **Progress Display**: Show progress for each challenge
- **Cooldown Timers**: Display remaining cooldown
- **Reward Information**: Show XP rewards
- **Click to View**: Click for detailed information

### Claim Map GUI
- **Visual Map**: 11x11 chunk map
- **Color Coding**: Different colors for kingdoms
- **Interactive**: Click chunks to teleport
- **Information**: Hover for chunk info
- **Navigation**: Navigate map

### Statistics Dashboard
- **Comprehensive Stats**: All kingdom statistics
- **Visual Charts**: Progress bars and graphs
- **Historical Data**: Growth over time
- **Comparisons**: Compare with other kingdoms

### Territory Heat Map
- **Activity Visualization**: Color-coded activity
- **Intensity Display**: Show activity intensity
- **Time Range**: Select time range
- **Interactive**: Click for details

### Mail GUI
- **Inbox View**: List all mail
- **Read/Unread**: Visual indicators
- **Mail Reading**: Click to read
- **Mail Management**: Delete, mark as read

### Event Calendar
- **Monthly View**: Calendar interface
- **Event Display**: Show events on days
- **Navigation**: Previous/next month
- **Event Details**: Click for event info

### Vault GUI
- **Item Storage**: Store items
- **Access Control**: Permission-based
- **Item Management**: Add/remove items

## Animated GUIs

### Progress Bars
- Visual progress indicators
- Percentage display
- Color-coded

### Animated Borders
- Rotating border materials
- Visual effects
- Particle effects

## GUI Interactions

### Click Handlers
- Left click for actions
- Right click for info
- Shift click for admin

### Navigation
- Previous/next pages
- Back buttons
- Close buttons
